kb

Kanban in your terminal

A portable CLI Kanban board that lives in your terminal and opens a web dashboard when you need visuals. Written in Rust — one binary, zero dependencies.

Get Started See it in action
kb demo

Add tasks, assign them, move columns, open the dashboard — all from the terminal.

Why kb?

▸

CLI-First

Manage tasks without leaving your terminal. Fast, scriptable, composable — fits naturally into your dev workflow.

▸

Web Dashboard

Drag-and-drop board served from your machine. No cloud, no signup, no data leaving your computer.

▸

Single Rust Binary

One file, zero runtime dependencies. Works on Windows, Linux, and macOS. ~5MB on disk.

▸

Markdown Native

Your board is a Markdown file with YAML frontmatter. Version control friendly, human readable, easy to edit.

Get kb up and running in seconds.
