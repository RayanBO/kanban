kb

Kanban in your terminal

A portable CLI Kanban board that lives in your terminal and opens a web dashboard when you need visuals. Labels, user assignment, and watch mode are built in. Written in Rust — one binary, zero dependencies.

Get Started View Docs See it in action
kb demo

Add tasks, assign them, tag them, move columns, and open the dashboard — all from the terminal.

Why kb?

▸

CLI-First

Manage tasks without leaving your terminal. Fast, scriptable, composable — fits naturally into your dev workflow.

▸

Web Dashboard

Drag-and-drop board served from your machine with live reload and label filters. No cloud, no signup, no data leaving your computer.

▸

Single Rust Binary

One file, zero runtime dependencies. Works on Windows, Linux, and macOS. ~5MB on disk.

▸

Tags & Due Dates

Label tasks with tags, set due dates, and filter by priority, status, or labels. The dashboard has a tag-based filter and a collapsible task panel.

▸

Comments & Export

Add Markdown comments to any task, export your board as JSON or Markdown, and track changes with full comment history.
